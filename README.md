# Tic Tac Toe Pro: Ultimate Edition 🎮

A premium, cross-platform Tic Tac Toe application built with Flutter. This project features an unbeatable AI, dynamic animations, and a professional web-inspired dashboard.

## ✨ Features

- **Multiple Game Modes**: 
  - **PvP**: Play locally with a friend.
  - **PvE**: Challenge a smart AI with 3 difficulty levels (Easy, Medium, Hard).
- **Intelligent Hint System**: 
  - Get strategic move suggestions when you're stuck.
  - Limits based on difficulty: Unlimited (Easy), 3 (Medium), 1 (Hard).
- **Modern Dashboard UI**: 
  - **Responsive Layout**: Column-based dashboard for desktop/web and compact view for mobile.
  - **Move History**: Real-time log of every move made in the game.
  - **Last Move Preview**: Mini-map highlighting the most recent interaction.
- **Premium Aesthetics**:
  - **Animated Background**: Floating, rotating shapes (triangles, circles, squares) for an attractive "Dreamy" feel.
  - **Dark & Light Modes**: Fully adaptive themes with a floating toggle.
  - **Zero Scrollbars**: Custom scroll behavior for a clean, professional web look.
- **Game Mechanics**:
  - **Turn Timer**: 20-second countdown to keep the game fast-paced.
  - **Dynamic Board Scaling**: Supports 3x3, 4x4, and 5x5 grid sizes.
  - **Score Tracking**: Keep count of wins for both players.

## 🚀 Getting Started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your machine.
- A code editor (VS Code, Android Studio, or IntelliJ).

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/tictactoe-ultimate.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd tictactoe-ultimate
   ```

3. **Install dependencies**:
   ```bash
   flutter pub get
   ```

4. **Run the app**:
   ```bash
   flutter run
   ```
